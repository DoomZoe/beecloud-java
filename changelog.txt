BeeCloud Java Changelog
============================

v2.3.0 发布日期：2015年9月27日
+ 新增百度钱包支付，包括支付、退款、订单查询、退款查询。
+ 新增易宝点卡支付，包括支付、订单查询。
+ 新增根据订单记录或者退款记录ID查询订单或者退款单记录。
+ 支持预退款，新增needApproval字段表示直接退款或预退款。
+ 支持订单超时设置，新增billTimeout字段设置订单超时时间。
+ 新增WX JSAPI支付的demo实现。
+ 重构SDK API参数，传入参数由一个封装类取代多个参数。

v2.2.0 发布日期：2015年9月1日
+ 添加京东，易宝，快钱三个新渠道
+ 包括京东网页支付，京东wap支付，易宝网页支付，易宝wap支付，快钱网页支付，快钱wap支付

v2.1.0 发布日期：2015年8月26日
+ 新增阿里批量打款
+ 修改退款、订单查询、退款查询 channel为选填

v2.0.0 发布日期：2015年7月27日
+ 随机选取后台api服务器实现负载均衡
+ ５个rest api 接口函数，支持各种渠道。接口统一，完善
+ 包含支付，退款，订单查询，退款查询，微信退款状态查询等5项基本功能
+ 基于Jsp实现的demo，包括同步通知return_url, front_url以及webhook异步通知notify_url的实现样例
+ 支持Maven配置依赖
+ 包含Javadoc生成的doc文件夹

v1.4 发布日期：2015年6月30日
+ 增加微信、支付宝、银联交易订单查询功能
+ 增加微信、支付宝、银联预退款和确认退款功能
+ 增加微信退款查询功能
+ 修复支付参数检查bug

v1.3.2 发布日期：2015年6月24日
+ 优化服务器最优选取策略
+ 支付返回参数统一变成BCPayResult

v1.3.1 发布日期：2015年5月12日
+ 寻找服务器最优策略
+ 出错服务器重连bug

v1.3 发布日期：2015年4月17日
+ 新增微信红包接口

v1.2 发布日期：2015年4月13日
新增银联在线支付

v1.1 发布日期：2015年4月1日
+ 优化最优服务器匹配，提高服务稳定性
+ 新增微信扫码

v1.0 发布日期：2015年3月20日
+ BeeCloud秒支付支付宝支付发布
+ BeeCloud云数据库功能发布
+ BeeCloud地理信息功能发布
